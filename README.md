## Maps Of Republic Indonesia
A project that using Leaflet dan ArcGIS Services to build a map of the Republic of Indonesia. Data sourced from legal open data sources published by National Geospatial Agency.

Contains information about Republic of Indonesia Boundaries from legal public data.

Feel free to ask the details or you want to collaboration for sharing similiar projects.

## Features
- **Supporting for One Map Policy** - Using data from One Map Policy that published by National Geospatial Agency.
- **Multiple Maps Services** - Using several ArcGIS REST Services and compile it into one map theme.
- **Interactive Maps** - Using Leaflet and It's plugins to build some features for interactive maps.
- **Basemap Options** - Two options for viewing the maps with OSM maps and ESRI Basemaps (Imagery).
- **Layer Selection** - Able to select which layer will be showed on the maps.
- **Responsive Legend** - Legend that showing only active layers.
- **Informative Popups** - Popups that contain short information the bpundaries.
- **Portofolio and Contacts Button** - Easy Access for anyone who interesting to know some details about myself if interesting.

## Disclaimer
The visibility data depends on the National Geospatial Agency server.
